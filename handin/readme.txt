phproot.tar.gz
Compressed file containing all PHP programs and HTML files needed to run the pallet maintenance system.

HOW TO TEST SYSTEM:
Unpack the package phproot.tar.gz.
Start the built-in web server:
cd phproot
php -S 127.0.0.1:8080
Open your web browser and go to:
	http://localhost:8080/index.html
				OR
	http://127.0.0.1:8080/index.html
